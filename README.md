# Bilibili 视频数据分析

## 项目简介

本项目基于从B站公开API采集的约 **9,000 条** 视频数据，涵盖多个分区（小剧场、动物、美食、知识、音乐等），对视频的播放量、互动率、发布时间、UP主影响力等维度进行了深度分析，并使用 **FineBI** 制作了可视化仪表板，旨在为内容创作者和平台运营提供数据驱动的决策参考。

## 数据来源

- 采集方式：Python 爬虫（B站分区推荐接口 `/x/web-interface/region/feed/rcmd`）
- 数据时间：2026年4月（近期数据）
- 原始字段：
  - 标题、作者、分区
  - 播放数、弹幕数、点赞数、硬币数、收藏数、转发数
  - 发布时间、视频时长

## 数据结构

清洗后的数据包含以下字段（示例）：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 标题 | str | 视频标题 |
| 作者 | str | UP主名称 |
| 播放数 | int | 播放量 |
| 弹幕数 | int | 弹幕数量 |
| 点赞数 | int | 点赞数 |
| 硬币数 | int | 投币数 |
| 收藏数 | int | 收藏数 |
| 转发数 | int | 转发数 |
| 发布时间 | datetime | 发布日期时间 |
| 时长 | str | 视频时长（HH:MM:SS） |
| 分区 | str | 所属分区（如美食、科技数码等） |

> 注：时长字段已转换为秒数 `duration_sec` 并划分时长分组。

## 分析内容

1. **整体数据概览**  
   - 总视频数、总播放量、平均播放量、平均互动率

2. **分区表现分析**  
   - 各分区视频数量分布  
   - 各分区平均播放量  
   - 各分区平均互动率  
   - 各分区播放量与收藏数对比

3. **UP主分析**  
   - 播放量 TOP10 UP主

4. **播放量与转发数关系**  
   - 散点图识别“高转发低播放”潜力视频

5. **发布时间分析**  
   - 各小时平均播放量趋势，找出最佳发布时间

6. **时长分析**  
   - 不同时长分组的播放量分布（0-1min、1-5min、5-10min、>10min）

## 可视化工具

- **FineBI**：制作交互式仪表板，包含 KPI 卡片、条形图、散点图、折线图等组件
